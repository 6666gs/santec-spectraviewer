## 公式引擎 (`_eval_formula`)

### 语法
```
A0, A1, A2, ...  # 引用表格中对应索引的光谱
+, -, *, /, **   # 算术运算
np.sin, np.log   # numpy 函数
```

### 示例
```
A12 - A1              # 差分
A1 - (A2 - A1) * 2    # 线性基线扣除
-10 * np.log10(A1)    # dBm → mW 转换
```

### 处理流程

1. 提取公式中所有 `A{n}` 索引
2. 加载涉及的光谱数据
3. 计算各数据 x 范围的交集
4. 将所有 y 插值到公共网格
5. 在 numpy 命名空间中 `eval()`

### 错误处理
- 索引越界 → 提示有效范围
- 无交集 → 提示"各数据的 x 范围无交集"
- 语法错误 → 显示异常信息

## 峰值/谷值分析

### 输入参数
| 字段 | 说明 |
|------|------|
| 搜索 X 范围 | 限定分析区间 |
| 阈值 (dB) | 峰/谷高度阈值 |
| 最小间距 | 防止噪声误检 |

### 输出
- 终端打印每个峰的位置、值、3dB 带宽
- 图像标注峰位置和信息

### 3dB 带宽计算
- **峰值**：峰顶向下 3dB 的宽度
- **谷值**：谷两侧参考电平向下 3dB 的宽度

## 微环分析面板

### 参数
| 字段 | 说明 |
|------|------|
| 波长范围 (nm) | FSR 和 Q 分析的范围 |
| 显示单峰拟合 | 是否弹出单峰洛伦兹拟合图 |

### 流程
```
点击[微环分析]
    ↓
Ring(x, y)
    ↓
cal_fsr(range_nm) → 显示 FSR 图
    ↓
cal_Q(holdon, max_holdon=10) → 显示 Q 分布图
    ↓
终端输出: FSR均值, Ql均值, Qi均值
```
